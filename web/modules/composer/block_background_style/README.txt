INTRODUCTION
------------

Block Background Style Module adds the background image and background style 
to the blocks.Users can upload the background image for each block in the 
block configuration page and also add css for each block from the 
configuration page .

 * For a full description of the module, visit the project page:
   https://www.drupal.org/project/block_background_style

 * To submit bug reports and feature suggestions, or track changes:
   https://www.drupal.org/project/issues/block_background_style

REQUIREMENTS
------------

This module requires no modules outside of Drupal core.

 

INSTALLATION
------------

 * Install as you would normally install a contributed Drupal module. Visit
   https://www.drupal.org/node/1897420 for further information.

CONFIGURATION
-------------

The module has no configuration settings.
